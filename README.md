# Student Growth Index (SGI)

A PHP web application to track and calculate student academic growth using MongoDB Atlas.

## Setup

1. Clone the repo
2. Run `composer install`
3. Copy `cacert.pem` from your PHP installation into the project root
4. Update `config.php` with your MongoDB Atlas connection string
5. Run on a PHP 8+ server (XAMPP locally)

## Tech Stack
- PHP 8+
- MongoDB Atlas
- Chart.js
